1-D Monocopter PID controller simulator Original page (Korean)
  • Mass Err : Mass the controller perceives. If greater than 1, controller gives more thrust and monocopter flies higher.
  • P Value : P value of PID controller (0-50)
  • D Value : D value of PID controller (0-20)
  • I Value : I value of PID controller (0-20)
  • INIT button : Initialize
  • RUN button : Run simulator
